개인정보 처리방침

최종 수정일: 2026년 2월 18일

이 개인정보 처리방침은 BTCT/DOGE DEX 플랫폼 및 BTCT Town을 이용할 때 수집되는 데이터, 이용 방법, 보호 방법을 설명합니다.

1. 플랫폼 성격

이 플랫폼은 비수탁형(Non-Custodial) 오픈소스 탈중앙화 거래소(DEX)로, BTCT와 DOGE 간의 P2P HTLC 아토믹 스왑을 중계합니다. 서버는 오직 게시판 및 메시지 중계자 역할만 수행합니다.

서버는 개인키나 사용자 자산을 보유·접근·통제·저장하지 않습니다.

2. 수집하는 데이터

데이터 종류 설명 목적
BTCT 주소 공개 블록체인 주소 (개인정보 아님) 거래 조율, 게시물 식별
DOGE 주소 공개 블록체인 주소 거래 조율 (DOGE 전송 대상)
거래 데이터 광고 상세 정보, 거래 상태, HTLC 해시 아토믹 스왑 조율
채팅 메시지 거래 참여자 간 텍스트 메시지 (최대 500자) 거래 소통, 분쟁 참조
타운 위치 인게임 캐릭터 좌표 (BTCT Town 한정) 멀티플레이어 기능 (DB에 저장 안 됨)

3. 수집하지 않는 데이터

  • 개인키 — 모든 서명은 사용자의 브라우저에서만 처리됩니다
  • 개인정보 — 이름, 이메일, 전화번호, 신분증, KYC 정보 없음
  • IP 주소 — 애플리케이션에서 기록·저장하지 않습니다
  • 쿠키 및 추적 — 분석 도구 없음, 제3자 트래킹 없음
  • 브라우저 핑거프린트 — 수집하지 않습니다

4. 클라이언트 저장소 (localStorage)

아래 데이터는 브라우저의 localStorage에만 저장되며, 서버로 전송되지 않습니다:

  • 지갑 주소 및 개인키 (브라우저 내 보관)
  • HTLC 시크릿 (아토믹 스왑을 위해 로컬에서 생성)
  • 면책 동의 여부

⚠ 브라우저 데이터를 초기화하면 로컬에 저장된 지갑 키와 거래 시크릿을 잃게 됩니다. 반드시 개인키를 별도로 백업해 두세요.

5. 채팅 메시지 정책

  • 채팅은 거래 참여자(매도자·매수자)끼리만 이용 가능합니다 (1:1, 거래별).
  • 서버는 메시지 저장·전달 전에 발신자가 해당 거래의 참여자인지 검증합니다.
  • 메시지는 최대 500자로 제한됩니다.
  • 위협·불법 내용 등 금지 키워드가 포함된 메시지는 자동 차단됩니다.
  • 채팅 메시지는 생성일로부터 90일 후 자동 삭제됩니다.

6. 데이터 보존 기간

데이터 종류 보존 기간
거래 게시물(광고) 무기한 (공개 게시판)
거래 기록 무기한 (블록체인 조율 이력)
채팅 메시지 90일 (자동 삭제)
타운 플레이어 위치 세션 동안만 (인메모리, 접속 해제 시 소멸)

7. 데이터 공유

귀하의 데이터는 제3자와 공유되지 않습니다. 사용되는 유일한 외부 서비스:

  • Blockcypher API — DOGE 잔액 조회 및 트랜잭션 브로드캐스트 용도. 공개 블록체인 데이터(주소, 트랜잭션)만 전송되며, 개인 데이터는 공유되지 않습니다.

8. 보안 조치

  • 모든 개인키 처리는 클라이언트(브라우저)에서만 이루어집니다
  • 서버 접근은 UFW 방화벽, Fail2Ban, 요청 속도 제한으로 보호됩니다
  • API 속도 제한: 지갑당 24시간 내 BTCT 50,000개, 스왑 30회, 광고 등록 20회
  • SSH 강화 (Fail2Ban, 접속 세션 제한)
  • 데이터베이스 접근은 로컬호스트로만 제한

9. 이용자 권리

개인정보를 수집하지 않으므로, 전통적인 정보주체 권리(열람·삭제·이동)는 다음과 같이 제한적으로 적용됩니다:

  • 지갑 데이터는 브라우저에만 저장되어 있으며 이용자가 완전히 통제합니다.
  • 거래 기록은 공개 블록체인 주소에 연결된 조율 데이터로, 개인 식별자가 아닙니다.
  • 채팅 메시지는 90일 후 자동 삭제됩니다.
