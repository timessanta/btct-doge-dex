1 이게 뭔가요?

🤝 중개자 없이 직접 교환

이 거래소는 BTCT(Bitcoin Time)와 DOGE(Dogecoin)를 서로 교환하는 서비스입니다.

일반 거래소와 다르게 서버가 여러분의 코인을 보관하지 않습니다. 거래 상대방과 스마트 컨트랙트를 통해 직접 교환되기 때문에, 서버가 해킹당해도 여러분의 코인은 안전합니다.

🔐 어떻게 안전하게 교환되나요?

"비밀번호 열쇠" 방식을 사용합니다. 판매자가 비밀 열쇠를 만들고, 그 열쇠로만 풀 수 있는 자물쇠에 코인을 잠급니다. 구매자도 같은 자물쇠에 DOGE를 잠그면, 판매자가 열쇠를 사용해 DOGE를 가져올 때 열쇠가 자동으로 공개되고 구매자도 BTCT를 가져갈 수 있게 됩니다.

이 방식을 아토믹 스왑(Atomic Swap)이라고 합니다. 양쪽 모두 가져가거나, 아무도 못 가져가거나 — 중간은 없습니다.

💡 이 거래소는 오픈소스입니다. 누구나 코드를 확인하고 직접 실행할 수 있습니다. → GitHub 보기
2 지갑 준비하기

🔑 BTCT 지갑

  1. 상단 메뉴 Wallet 탭 클릭
  2. "New BTCT Wallet" 클릭 → 새 지갑 생성
  3. 화면에 표시된 개인키(Private Key)를 안전한 곳에 저장
    ※ 잃어버리면 복구 불가합니다
  4. 우측 상단에 지갑 주소(0x로 시작)가 표시되면 연결 완료
⚠️ 개인키는 절대 타인에게 공개하지 마세요. 개인키를 알면 코인을 모두 가져갈 수 있습니다.

🐕 DOGE 지갑

  1. Wallet 탭 → DOGE 섹션에서 "New DOGE Wallet" 클릭
  2. 생성된 DOGE 주소(D로 시작)와 WIF 개인키를 저장
  3. 거래 전에 해당 주소로 DOGE를 충전해 두세요

🔄 기존 지갑 불러오기

  1. Wallet 탭 → "Import" 클릭
  2. 저장해 둔 개인키 / WIF 붙여넣기 → 불러오기 완료
💡 지갑 정보는 이 브라우저에만 저장됩니다. 다른 기기에서 사용하려면 개인키를 직접 불러와야 합니다.
3 BTCT 팔기 (DOGE 받기)

📋 광고 등록하기

  1. 상단 메뉴 Bulletin Board 클릭
  2. "Post Listing" 클릭
  3. 타입: SELL BTCT 선택
  4. 가격 입력 (1 BTCT = ? DOGE)
  5. 최소/최대 BTCT 수량 입력
  6. Post 버튼 클릭 → 게시판에 광고 등록 완료
💡 광고가 등록되면 BTCT Town에서 내 캐릭터 머리 위에 말풍선이 표시됩니다.

⏳ 거래 요청이 오면?

구매자가 광고를 보고 거래를 요청하면 알림이 옵니다. My Listings 또는 Swap History에서 거래 번호를 클릭해 진행하세요.

거래가 시작되면 5단계 스왑 진행이 필요합니다. → 5단계 흐름 보기

4 BTCT 사기 (DOGE 주기)

🔍 원하는 광고 찾기

  1. Bulletin Board에서 SELL 광고 목록 확인
  2. 원하는 광고의 "Start Swap" 클릭
  3. 구매할 BTCT 수량 입력 후 확인

📤 거래 진행

거래가 생성되면 판매자가 먼저 시작합니다. 알림을 받으면 Swap History에서 해당 거래를 열어 각 단계를 진행하세요.

⚠️ DOGE 지갑에 충분한 DOGE가 있어야 합니다. 거래 시작 전에 잔액을 확인하세요.
5 스왑 5단계 전체 흐름

각 단계에서 버튼을 클릭하면 지갑 서명이 이루어집니다. 서버는 서명에 관여하지 않으며, 모든 서명은 이 브라우저에서만 처리됩니다.

1
판매자

비밀 열쇠 생성 & 공개

판매자가 무작위 비밀번호(secret)를 만들고, 그 해시(잠금 암호)를 거래에 등록합니다. 비밀번호 원본은 판매자만 알고 있습니다.

2
판매자

BTCT를 스마트 컨트랙트에 잠금 (약 2시간)

판매자가 팔 BTCT를 "자물쇠 계좌(HTLC)"에 전송합니다. 비밀번호를 아는 사람만 가져갈 수 있고, 약 2시간 후에는 판매자가 환불받을 수 있습니다.

3
구매자

DOGE를 스마트 컨트랙트에 잠금 (약 1시간)

구매자가 지불할 DOGE를 또 다른 자물쇠 계좌에 전송합니다. 판매자가 비밀번호를 공개해야만 가져갈 수 있고, 약 1시간 후에는 구매자가 환불받을 수 있습니다.

4
판매자

비밀번호 공개 & DOGE 수령

판매자가 비밀번호를 공개하며 DOGE를 가져갑니다. 이 순간 비밀번호가 블록체인에 기록되어 누구나 볼 수 있게 됩니다.

5
구매자

공개된 비밀번호로 BTCT 수령

구매자가 블록체인에서 비밀번호를 확인하고 BTCT 잠금 계좌에서 BTCT를 인출합니다. 거래 완료! 🎉

💡 왜 안전한가요? 4단계에서 판매자가 DOGE를 가져가려면 반드시 비밀번호를 공개해야 합니다. 그 즉시 구매자도 BTCT를 가져갈 수 있습니다. 한쪽만 손해보는 상황은 구조적으로 불가능합니다.
6 거래가 취소되면?

⏰ 시간이 지나면 어떻게 되나요?

상대방이 응답하지 않거나 중간에 취소된 경우, 설정된 시간이 지나면 자동으로 "만료(Expired)" 상태가 됩니다.

  • BTCT 환불: 약 2시간 후 판매자가 "Refund BTCT" 버튼 클릭 → 자동 환불
  • DOGE 환불: 약 1시간 후 구매자가 "Refund DOGE" 버튼 클릭 → 자동 환불

🔔 만료 알림

서버가 5분마다 자동으로 만료된 거래를 감지하여 알림을 보냅니다. 거래 페이지에 접속하면 환불 버튼이 자동으로 표시됩니다.

🚨 만료 전에는 절대 환불되지 않습니다. 타임아웃 시간 전에는 코인이 스마트 컨트랙트에 안전하게 잠겨 있습니다.
7 BTCT Town 이용하기

🎮 BTCT Town이란?

거래소에 연결된 픽셀아트 RPG 마을입니다. 실시간으로 다른 사용자들과 같은 공간에서 움직이며 거래할 수 있습니다.

🕹️ 기본 조작

  • PC: 방향키 또는 WASD로 이동
  • 모바일: 왼쪽 하단 조이스틱으로 이동
  • NPC(게시판) 접근: 가까이 가서 스페이스바 또는 ACT 버튼
  • 집 접근: 집 문 앞에서 스페이스바 → 캐릭터 꾸미기

📦 Town에서 거래하기

  1. NPC(게시판) 앞에서 스페이스바
  2. Post Listing → 광고 등록
  3. 다른 플레이어의 말풍선을 보고 관심 있는 상대에게 다가가서 거래 시작
  4. 거래 진행도 Town 내에서 바로 가능

🎨 캐릭터 꾸미기

  • 집 문 앞 접근 → 스페이스바 또는 우측 상단 🎨 버튼
  • 타입 (기본/전사/마법사/도적), 피부색, 옷 색상, 머리색, 모자, 안경 선택 가능
  • 변경 사항은 실시간으로 다른 플레이어에게도 보입니다

⛏️ 브라우저에서 BTCT 채굴하기

Town을 벗어나지 않고 브라우저에서 직접 BTCT를 채굴할 수 있습니다. 맵 상단에 ⛏️ Mine으로 표시된 채굴장에 접근하세요.

  1. Town 맵 상단의 채굴장으로 이동
  2. 스페이스바 또는 ACT 버튼으로 채굴 패널 열기
  3. 스레드 수 설정 (스레드 많을수록 해시레이트↑, CPU 사용량↑)
  4. Start Mining 클릭 → pool.btc-time.com 풀에 자동 연결
  5. 패널을 닫아도 채굴은 백그라운드에서 계속됩니다
  6. 채굴 중에는 상단 바의 ⛏️ 버튼이 깜빡입니다
  7. 매시간 정각에 풀에서 내 BTCT 주소로 자동 지급됩니다
⚠️ 브라우저 탭을 닫거나 새로고침하면 채굴이 중단됩니다.
8 자주 묻는 질문
가입이나 본인인증이 필요한가요? ▼
전혀 필요 없습니다. 지갑 주소가 곧 계정입니다. 이메일, 비밀번호, 개인정보 없이 바로 사용합니다.
서버가 코인을 가져갈 수 있나요? ▼
불가능합니다. 서버는 개인키를 전혀 가지고 있지 않습니다. 서명은 100% 이 브라우저에서만 이루어지며, 서버는 단순히 정보를 중계하는 역할만 합니다.
거래 수수료는 얼마인가요? ▼
별도 거래 수수료는 없습니다. 다만 블록체인에 트랜잭션을 보낼 때 소액의 네트워크 수수료(가스비)가 자동으로 차감됩니다. BTCT 수수료는 극히 소액, DOGE는 약 0.02 DOGE 수준입니다.
거래 상대가 취소하면 코인을 잃게 되나요? ▼
아닙니다. 코인은 스마트 컨트랙트(자물쇠 계좌)에 잠겨 있으며, 타임아웃 시간 후 환불 버튼으로 직접 돌려받을 수 있습니다. 상대방이 취소해도 여러분의 코인은 항상 안전합니다.
개인키를 잃어버렸어요. 어떻게 하나요? ▼
안타깝게도 복구가 불가능합니다. 개인키는 서버에 저장되지 않으며, 오직 본인만이 보관합니다. 반드시 안전한 곳에 백업해 두세요.
BTCT는 어디서 구하나요? ▼
이 거래소에서 DOGE로 구입할 수 있습니다. 또는 btc-time.com에서 직접 BTCT에 대해 알아보세요.
9 용어 정리
🔐 아토믹 스왑 (Atomic Swap)
두 코인을 동시에 교환하는 기술. 양쪽이 모두 성공하거나, 아무것도 진행되지 않습니다. "원자"처럼 쪼갤 수 없는 거래입니다.
🔑 개인키 (Private Key)
지갑의 비밀번호. 이걸 알면 그 지갑의 코인을 마음대로 移动할 수 있습니다. 절대 공개하면 안 됩니다.
📦 HTLC
Hash Time-Locked Contract. "자물쇠 계좌"입니다. 비밀번호를 알아야 코인을 꺼낼 수 있고, 시간이 지나면 주인이 환불받을 수 있습니다.
🏦 비수탁형 (Non-custodial)
거래소가 코인을 보관하지 않는 방식. 내 코인은 항상 내 지갑에 있습니다.
⛓️ 블록체인
거래 기록이 전 세계에 분산 저장된 공개 장부. 누구나 확인할 수 있고, 한 번 기록되면 수정이 불가능합니다.
🔤 해시 (Hash)
어떤 정보를 고정된 길이의 암호로 변환한 것. 원본에서 해시는 쉽게 만들 수 있지만, 해시에서 원본은 알 수 없습니다.
📡 트랜잭션 (Transaction)
블록체인에 기록되는 코인 이동 내역. 한 번 처리되면 변경할 수 없습니다.
⏰ 타임아웃
일정 시간이 지나도 거래가 완료되지 않을 경우 자동으로 종료되는 시간입니다. BTCT 약 2시간, DOGE 약 1시간.

이제 시작할 준비가 됐나요?

지갑을 만들고 첫 거래를 시작해 보세요.
